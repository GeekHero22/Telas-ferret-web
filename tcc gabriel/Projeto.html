Java Game


Vamos criar um projeto em java, então acho que não preciso explicar que preciso de um
programador além de mim kkkk, precisamos de pelo menos um desenhista que manje de arte digital
o jogo será um rpg de turno no estilo chrono trigger, só que com artes cartoon e não em pixel
precisaremos de alguém com experiência em roteiro para me ajudar a criar a história que se irá se
se passar em um universo baseado em shadowrun, misturando elementos ciberpunk e fántasticos, o jogador será
um lider de campo que encontrará outros mercenarios para ajudar ele na missão de derrubar o vilão, usaremos
java de inicio mas podemos mesclar com outras linguagens como c# no unity.

Vagas abertas no projeto


  • Arte digital

  • Programação em Java

  • Roteiro e criação de histórias
